## Instalação

Use o [git](https://git-scm.com/) para clonar o repositório

Entre na pasta do projeto e execute os comandos abaixo:

```bash
npm install # npm

yarn install # yarn
```

## Utilização

```bash
npm run start # npm

yarn start # yarn
```

## Build

```bash
# certifique-se que tenha as dependências instaladas (npm install)

npm run start # npm

yarn build # yarn

```
